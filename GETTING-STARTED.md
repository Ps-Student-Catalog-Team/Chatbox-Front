## 👋 欢迎！

---

## ⏱️ 5 分钟快速开始

### 步骤 1：启动应用（1 分钟）

打开终端，进入项目目录：

```bash
cd d:\项目\Chatbox-Front
go run main.go
```

你会看到：
```
服务启动在 :40001
数据库已初始化...
```

### 步骤 2：打开浏览器（1 分钟）

访问：
```
http://127.0.0.1:40001/index.html
```

### 步骤 3：登录（1 分钟）

- **账户**: 输入任意用户名（如：`testuser`）
- **密码**: 输入任意密码（如：`123456`）
- 点击 **登录** 或 **注册**

### 步骤 4：找到 AI 助手（1 分钟）

在左侧聊天列表中，你会看到：

```
最近会话
├── 公共聊天室
├── 🤖 Gemini 助手    ← 点击这个！
├── ... 其他
```

点击 **🤖 Gemini 助手**

### 步骤 5：测试对话（1 分钟）

在下方输入框输入：

```
你好，我是新用户。请自我介绍一下。
```

按 **发送** 按钮，等待 AI 回复（5-15 秒）

✅ **完成！** 你已经成功使用了 AI 聊天机器人！

## 🔧 常见任务

### 如何...

#### ...改变 AI 的性格？

编辑文件 `ai-chat.js`，找到第 14 行：

```javascript
SYSTEM_PROMPT: "你是一个神人ai..." // ← 修改这里
```

改为你想要的风格，如：

```javascript
// 专业顾问
SYSTEM_PROMPT: "你是一位经验丰富的技术顾问..."

// 热心朋友
SYSTEM_PROMPT: "你是一位热心的朋友..."

// 幽默达人
SYSTEM_PROMPT: "你是一位爱开玩笑的编程达人..."
```

重新刷新浏览器即可生效。

#### ...获取更多消息记录？

使用浏览器开发工具：

```javascript
// 按 F12 打开控制台，输入：
console.table(cacheMessages["ai:gemini"])

// 查看所有消息
```

#### ...导出聊天记录？

参考 [AI-QUICK-REFERENCE.md](AI-QUICK-REFERENCE.md) 的"高级功能"部分。

#### ...在手机上使用？

1. 确保手机和电脑在同一网络
2. 用手机浏览器访问：`http://YOUR_PC_IP:40001/index.html`
3. 按照相同步骤操作即可

## 🆘 遇到问题？

### 问题排查表

| 问题 | 症状 | 解决方案 |
|------|------|---------|
| 看不到 AI 项 | 左侧没有 🤖 | 刷新页面，查看 F12 控制台 |
| 无法发送消息 | 点击发送无反应 | 检查网络连接 |
| AI 无回复 | 消息被接收但没有回复 | 检查 API Key，等待更长时间 |
| 消息格式乱 | 文本显示异常 | 清空缓存，运行 `clearAIChatHistory()` |
| 性能卡顿 | 应用响应慢 | 清空对话记录，刷新页面 |

### 获取帮助

1. **查看文档**
   - [AI-CHAT-GUIDE.md](AI-CHAT-GUIDE.md) 的"故障排除"部分
   - [AI-QUICK-REFERENCE.md](AI-QUICK-REFERENCE.md) 的"常见问题速解"

2. **调试步骤**
   - 按 F12 打开开发者工具
   - 查看 Console 标签中的错误信息
   - 运行诊断命令

3. **重置应用**
   ```javascript
   // 在浏览器控制台运行
   clearAIChatHistory()  // 清空 AI 记录
   location.reload()     // 刷新页面
   ```

## 🎓 进阶使用

### 自定义 API Key

如果你有自己的 Gemini API Key：

1. 打开 `ai-chat.js`
2. 找到第 10-15 行的 `GEMINI_CONFIG`
3. 替换 `API_KEY` 值
4. 保存文件并刷新浏览器

### 扩展功能

在 `ai-chat.js` 中可以扩展：

- 添加文件上传支持
- 添加语音输入
- 添加代码执行
- 添加多语言支持

参考 [AI-CHAT-GUIDE.md](AI-CHAT-GUIDE.md) 的"高级用法"部分。

## 📊 性能和容量

| 指标 | 值 | 备注 |
|------|-----|------|
| 加载时间 | < 1 秒 | 首次初始化 |
| 消息延迟 | < 100ms | 本地处理 |
| API 响应 | 5-15 秒 | 取决于网络 |
| 缓存大小 | ~20MB | 100+ 消息 |
| 并发支持 | 多个浏览器 | 独立缓存 |

## 🔒 数据隐私

### 你的数据去哪了？

```
用户输入
    ↓
浏览器缓存（本地）
    ↓
发送到 Gemini API（Google）
    ↓
AI 处理和回复
    ↓
返回到浏览器
    ↓
显示和缓存
```

### 隐私建议

1. ✅ 定期清空对话记录
2. ✅ 不要发送个人敏感信息
3. ✅ 使用不同的 API Key 隔离
4. ✅ 在生产环境使用后端代理
