## 常见问题诊断

### ❌ 问题：看不到 "Gemini 助手" 项

**原因排查**：
1. 检查 `ai-chat.js` 是否已加载
   ```javascript
   // 在控制台运行
   typeof initializeAIChat // 应返回 "function"
   ```

2. 检查初始化是否完成
   ```javascript
   aiChatState.isInitialized // 应为 true
   ```

3. 查看控制台是否有错误

**解决方案**：
- 刷新页面
- 检查 `ai-chat.js` 路径是否正确
- 检查浏览器控制台错误

---

### ❌ 问题：发送消息后无回复

**原因排查**：
1. 检查网络连接
2. 检查 API Key 是否正确
3. 检查 API 配额

**在控制台运行**：
```javascript
// 手动测试 API
callGeminiAPI("你好").then(console.log).catch(console.error)
```

**解决方案**：
- 验证 API Key
- 检查网络连接
- 等待足够的时间（API 可能响应慢）

---

### ❌ 问题：收到 CORS 错误

**症状**：控制台显示 "No 'Access-Control-Allow-Origin' header"

**解决方案**：
1. 这是正常的（浏览器限制）
2. 如果 API 调用失败，可能需要使用后端代理

---

### ❌ 问题：消息格式错乱

**症状**：消息显示异常或乱码

**解决方案**：
1. 清空浏览器缓存
   ```javascript
   clearAIChatHistory()
   ```
2. 刷新页面
3. 清空本地存储
   ```javascript
   localStorage.clear()
   ```
