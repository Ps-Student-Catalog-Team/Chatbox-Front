# 新消息提醒升级说明

## 📋 更新内容

已改进新消息提醒功能，提供更直观的用户体验。

## 🎯 新功能

### 1. 📌 标题栏显示未读消息数

- **替代** ❌ 之前的闪烁标题
- **改为** ✅ 浏览器标签页显示未读消息计数
- **格式**: `(3) 原标题` - 直观显示未读数量

### 2. ↓ 回到最新消息按钮

- 聊天窗口右上角新增按钮
- 当向上滚动查看历史消息时显示
- 点击快速回到最新消息
- **标签**: "↓ 最新"

### 3. 🎯 新消息分隔线

- 在最旧的新消息**上方**显示分隔线
- 分隔线中间标注 **"新消息"** 文字
- 红色渐变线条，带动画效果
- 帮助快速定位新内容

### 4. 🎨 智能滚动体验

- 打开聊天窗口时自动清除新消息标记
- 点击"↓ 最新"按钮后清除标记
- 无新消息时正常滚动到底部
- 有新消息时自动定位到分隔线

## 📊 行为说明

### 场景 1：后台接收新消息

```
操作流程：
1. 用户在聊天室1，收到聊天室2的新消息
2. 效果：
   - 标题: "(1) 原标题"
   - 侧边栏聊天室2显示红色徽章 "1"
   - 通知+提示音
   
3. 用户点击聊天室2
4. 效果：
   - 自动定位到新消息分隔线
   - 分隔线上方是历史消息
   - 分隔线下方是新消息
   - 新消息标记清除，标题恢复正常
```

### 场景 2：向上浏览历史消息

```
操作流程：
1. 用户在聊天室1，有新消息
2. 用户向上滚动查看历史
3. 效果：
   - "↓ 最新"按钮出现
   - 浅蓝色高亮，带悬停效果
   
4. 用户点击"↓ 最新"
5. 效果：
   - 平滑滚动回到最新消息
   - 按钮隐藏
   - 新消息标记清除
```

### 场景 3：多个聊天的新消息

```
操作流程：
1. 收到多个聊天的新消息
2. 效果：
   - 标题: "(5) 原标题"
   - 每个聊天显示各自的未读数
   - 都有新消息分隔线
   
3. 打开任一聊天
4. 效果：
   - 该聊天的新消息标记清除
   - 标题更新为: "(3) 原标题"
   - 其他聊天的新消息保留
```

## ❓ 常见问题

### Q: 标题不显示未读数？

**A:** 检查设置是否启用了"标题提醒"功能：
- 打开聊天 → 右上角⋮ → 提醒设置
- 勾选"标题提醒"

### Q: "回到最新"按钮在哪里？

**A:** 聊天窗口右上角，在"刷新"按钮左边：
```
[↓ 最新] [🔄 刷新] [⋮]
```
向上滚动时自动显示。

### Q: 新消息分隔线不出现？

**A:** 这是正常现象：
- 打开聊天时，分隔线被清除
- 只有**后续收到的新消息**才显示分隔线
- 关闭聊天、再回来会再次出现

### Q: 如何完全禁用新消息标记？

**A:** 在浏览器控制台执行：
```javascript
// 清除新消息标记
if (activeTarget) clearNewMessageMarks();

// 重新渲染
renderChatBubbles();
```
